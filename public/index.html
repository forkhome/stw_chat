STW Chat (Star Wars Chat)

Another chat with node and socket.io

This is just another chat with nodejs and socket.io. I didn't have good name for it and wanted to name it just another node chat. But in order to demo it I took star wars heroes as our users. So welcome stw chat.

The aim of this project is to help node newbies to create chat. And personally for me this was just another opportunity to work with node.
